# TaskFlow – Assignment Manager

A modern full-stack assignment tracking web application built using Flask.

## 🚀 Features
- Add, delete assignments
- Mark as completed / pending
- Dashboard with stats
- Date & time tracking
- Clean blue UI design

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite

## 📌 How to Run
1. Install Flask
2. Run:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
